What is Somn?

Machine-learning methods have great potential to accelerate the identification of reaction conditions for chemical transformations. Somn has been developed to predict substrate-adaptive conditions for palladium (Pd)–catalyzed carbon-nitrogen (C–N) couplings.

Learn more about the model design and development, access training data , GitHub public code repository, and publications .
